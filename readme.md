# DS Registro Avançado com OTP

**Versão:** 2.1.0  
**Status:** ✅ TOTALMENTE FUNCIONAL  
**Última atualização:** 05/11/2025  

Sistema avançado de registro de usuários com verificação OTP via WhatsApp, desenvolvido pela DSantos Info.

## Descrição

Implementa fluxo completo de registro com verificação de telefone via WhatsApp e conclusão obrigatória de perfil.

## Funcionalidades

- Registro multi-etapa com Gravity Forms
- Verificação OTP via WhatsApp
- Conclusão forçada de perfil
- Integração com WooCommerce
- Mapeamento visual de campos
- Padronização de números de telefone
- Sistema de templates de mensagens

## Dependências

- Gravity Forms
- WooCommerce
- Conector WhatsApp DS Info

## Requisitos

- WordPress 5.0+
- PHP 8.0+

## Instalação

1. Instale as dependências necessárias
2. Faça upload do plugin para `/wp-content/plugins/`
3. Ative o plugin no painel do WordPress
4. Configure os formulários nas opções do plugin
